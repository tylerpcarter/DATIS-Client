Project Description

This web project serves as a full stack CRUD application implementing a RESTful API. A user is able to create, read, update, and delete employees and employee data out of a database. Each employee has information on current salary and past salaries, all bonuses for the current year, and all deductions for the current year. Finally, the application dynamically calculates the employee's take home pay.
